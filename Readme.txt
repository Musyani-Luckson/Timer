# Countdown Timer Web Application

This is a simple web application that allows users to set a countdown timer for a future date and time. The application displays the remaining time in days, hours, minutes, and seconds until the target date and time are reached.

## Features

- Set the target date and time for the countdown.
- Display the remaining time in days, hours, minutes, and seconds.
- Validate that the entered date and time are in the future.
- Update the countdown display in real-time.
- Show the name of the countdown event.
- Show the target date and time in a user-friendly format.

2. Open the `index.html` file in your web browser.

3. Use the input fields to enter the name, date, and time for the countdown event.

4. Click the "Update" button to start the countdown.

## Usage

1. Enter the name of the event in the "Event Name" input field.

2. Enter the date of the event in the "Date" input field. Make sure the date is in the future.

3. Enter the time of the event in the "Time" input field.

4. Click the "Update" button to start the countdown.

5. The remaining time in days, hours, minutes, and seconds will be displayed on the screen.

6. The countdown will update in real-time until the target date and time are reached.

## Examples

Here are some examples of valid input formats:

- Event Name: Birthday Party
- Date: 2023-12-31
- Time: 18:00

- Event Name: New Year's Eve
- Date: 2024-01-01
